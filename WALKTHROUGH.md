# Dress Up Game Walkthrough

## Overview
A web-based dress-up game featuring a premium glassmorphism UI, a central doll figure, and 10 categories of clothing/accessories.

## Features
- **10 Categories**: Hair, Hat, Top, Bottom, Acc, Shoes, Socks, Outer, Face, BG.
- **Drag & Drop**: Items can be moved freely on the doll.
- **Resizing**: Click an item to select it, then drag the bottom-right handle to resize.
- **Layering**: Items automatically layer correctly (e.g., Underwear < Top < Jacket).
- **Removal**: Select an item and click the 'x' button or press the "Reset All" button to clear everything.
- **Dynamic Panels**: Left and Right panels split the categories for better organization.

## How to Play
1. **Select a Category**: Click tabs on the left or right panels.
2. **Wear an Item**: Click an item in the grid to add it to the center stage.
3. **Adjust**:
    - **Move**: Drag the item on the doll.
    - **Resize**: Click the item to see the resize handle (blue dot), then drag it.
    - **Remove**: Click the 'x' on the selected item.

## Technical Details
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript.
- **Assets**: Generated using AI, with fallback colored placeholders for scalability.
- **Responsive**: Adapts to screen size (desktop focused).
